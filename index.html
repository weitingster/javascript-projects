Objectives

  • To one day share how a non-technical Product Manager, got technical

  • To keep track of my coding endeavours so I can always refer back to it

Many projects are inspired by 100+ Javascript Projects for Beginners. They built projects using libraries, with different designs. I built via pure Vanilla Javascript, tweaked designs, and without viewing their code.

Disclaimer: There are multiple ways to write the code. My projects may not be optimized for the most efficient way or the most prettiest way. It's a reminder to myself that I'm constantly learning and in the future, I'll look back and say "wow, I really need to refactor this because I know a better approach now."
